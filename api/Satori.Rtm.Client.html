Interfaces

IRtmClient

The IRtmClient interface is the main entry point for accessing the RTM service, including publish and subscribe operations.

ISubscription

Describes a subscription.

ISubscriptionEventSource

This interface describes subscription events. It is not meant to be implemented in user code. The SubscriptionObserver class is a default implementation of this interface. Use an instance of the SubscriptionObserver class to provide the IRtmClient.CreateSubscription(String, SubscriptionModes, ISubscriptionObserver) method with a subscription observer.

ISubscriptionObserver

Describes an observer interface which should be implemented in order to listen for subscription events. Use an instance of the SubscriptionObserver class to provide the IRtmClient.CreateSubscription(String, SubscriptionModes, ISubscriptionObserver) method with a subscription observer.
