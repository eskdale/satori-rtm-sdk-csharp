MaxReconnectInterval

Gets or sets the maximum time period to wait between reconnection attempts.

Declaration
public TimeSpan MaxReconnectInterval { get; set; }
Property Value
Type Description
System.TimeSpan
Remarks

The following formula is used to calculate the reconnect interval after the client disconnects from the RTM service for any reason:

Min(MinReconnectInterval * (2 ^ (attempt_number), MaxReconnectInterval)

The timeout period between each successive connection attempt increases, but never exceeds this value.
